Student Marks Calculator (.NET Console Application)

A simple C# console application that calculates a student's final course mark based on weighted assessments. The program validates user input, calculates the weighted average, and determines whether the student qualifies for the final exam.

✨ Features
Prompts the user for their name.
Accepts marks for:
Test 1 (30%)
Test 2 (50%)
Assignment (10%)
Project (10%)
Validates that all entered marks are between 0 and 100.
Calculates the weighted final mark.
Displays a summary of the entered marks.
Determines whether the student has passed (50% or higher) or failed.
🛠️ Technologies Used
C#
.NET Console Application
Visual Studio
📋 How It Works
Enter your name.
Enter the percentage for each assessment.
The application validates every mark before continuing.
The final weighted average is calculated using the following formula:
Assessment	Weight
Test 1	30%
Test 2	50%
Assignment	10%
Project	10%
The program displays:
The marks entered
The calculated final average
Whether the student passed or failed
